# Учебный проект «Шесть городов»(React, React Router, RTK)

Сервис поиска жилья 

### Описание 
В приложение несколько страниц, реализованы с помощью React Router. Имеется возможность сортировать предложения, добав-лять в избранное и оставлять комментарий.
Для управления состоянием и передачи данных использовала Redux Toolkit. Также реализована авторизация пользователя.
